### Проект представляет собой реализацию игры "виселица" с консольным вводом-выводом

Цель проекта: демонстрация знаний базового синтаксиса Python.

#### Процесс игры:
- Выбирается случайное слово из файла words.csv
- Случайным образом из слова убираются буквы

- Пользователь может совершить 6 ошибок.

- При проигрыше или выигрыше программа предлагает начать игру заново.

#### Особенности:
- наличие базовой валидации ввода пользователя,
- отрисовка игры ASCII-символами.
